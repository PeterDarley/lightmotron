Lightmotron

ESP32-S3

Install Lightmotron

MicroPython-based lighting controller for plastic model kits. Control NeoPixel LED strips with dynamic effects via a built-in web interface over WiFi.

Flash firmware to your ESP32-S3

Connect your ESP32-S3-DevKitC-1 via USB, then click the button below.

After installation: look for a WiFi network called lightmotron-setup on your phone or laptop. Connect to it and follow the on-screen instructions to enter your home WiFi credentials. The device will then be accessible at http://lightmotron.local/.

Requires Google Chrome or Microsoft Edge (Web Serial API). USB cable must support data transfer.

Hardware

  • ESP32-S3-DevKitC-1 (N16R8)
  • NeoPixel LED strip (default GPIO 4)
  • USB-C data cable

Browser

  • Google Chrome 89+
  • Microsoft Edge 89+
  • Web Serial API must be enabled

Features

  • NeoPixel strip lighting effects
  • Scenes, effects, filters, named ranges
  • Browser-based control interface
  • WiFi with captive-portal setup
  • Persistent configuration storage

Built-in effects

  • Solid, Blink, Pulse, Breathe
  • Wave, Cylon, Phaser Strip
  • Fade In, Scintillate, Sizzle
