Setup

Group LEDs into named ranges for use in animations and scenes.

  • Click an LED button to toggle it on or off.
  • Hold Shift and click a second LED to select or deselect the whole range between them.
  • Type a name and click Save to store the range.
  • Click an existing range name below the form to edit or delete it.
  • Click Clear to deselect all LEDs and start over.
